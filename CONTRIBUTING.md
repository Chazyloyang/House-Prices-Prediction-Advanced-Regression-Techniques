# Contributing to this Repository

First off, thank you for considering contributing! It's people like you that make open source and data science collaborative and great.

## Guidelines
1. **Reporting Bugs:** If you find a bug in the code or notebook, please open an Issue with a clear title and description.
2. **Suggesting Enhancements:** We are always open to improving our ML models or EDA. Feel free to open a Pull Request!
3. **Pull Requests:** Please make sure your code runs correctly before submitting a PR. Add clear commit messages explaining what you changed.

Happy coding!
